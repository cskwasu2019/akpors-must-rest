Get Resource Items

GET

@RESOURCE:

Type of resource, can be either of [jokes, motivations, proverbs]

@QUERY:

Optional Query
page: query resource items at specific page

RESPONSE:

Array of resource items data Object

EXAMPLES:

Get jokes resource (at page 1):
<%= package.homepage %>/api/jokes/
Get motivation resource (at page 3):
<%= package.homepage %>/api/motivations/?page=3
